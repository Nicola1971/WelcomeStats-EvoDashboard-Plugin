WelcomeStats OMWC Plugin for MODx Evolution v2.0 RC
===================

Dashboard MODX Stats widget for EvoDashboard and OnManagerWelcomeCustom plugin

**System Stat Counter for MODx Evolution**

* Documents
* Web Users
* Manager Users
* Admins
